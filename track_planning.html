SAT Competition 2020

Planning Track

The Planning Track is just like Main Track except for it has 200 benchmarks, which all come from the same application domain - automated planning.

The benchmarks will be in the standard DIMACS format (like in all the other tracks). We will select 100 satisfiable and 100 unsatisfiable formulas of various size and difficulty. All the formulas will encode either classical or HTN (Hierarchical Task Network) planning problem instances using several of the available encoding schemas.

If you wish to participate in the Planning Track you only need to register to the Main or Glucose-Hack Track, since all the participants of the those tracks are automatically included.
